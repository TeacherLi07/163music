# 163Music - 网易云音乐爬虫

## 功能列表

- [x] 从歌单id获取歌曲列表
- [x] 从歌曲id下载歌曲和歌词
- [x] 通过user agent，随机暂停，网络代理避免反爬虫
- [ ] 获取歌单详情
- [ ] 获取歌曲详情
- [ ] 将详情数据保存至数据库
- [ ] 添加CLI命令和界面

- [ ] 支持更多音乐平台
